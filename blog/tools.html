A list of free and open-source tools I've found over the years

02/01/2023

As a new developer, there were many obstacles in my way to create a new an amazing game. One of that obstacles was the lack of money, so in this post I'll be showing you some of the best softwares I've ever used in my developer career and I would've liked to know at the start of my journey.

Variety of multipurpose tools and softwares
  • Olive is a free and open-source cross-platform video editing application for Linux, Windows and macOS. It is currently in alpha.
  • DaVinci Resolve is a Blackmagic Design product and aims to deliver everything you need for video editing in a single program. It provides tools for editing images, adding visual effects and motion graphics, and audio processing.
  • Natron is a powerful Digital Compositor that can handle all of your 2D/2.5D needs. Its robust OIIO file formats and OpenFX architecture is what make Natron the most flexible open source compositor for the visual effects community.
  • ShareX is a free and open source program that lets you capture or record any area of your screen and share it with a single press of a key.
  • OBS Studio is a free and open source solution for offline video recording and live streaming that is Mac and Windows compliant. With an open canvas approach to video creation this tool can mix a variety of audio and video sources to a single output for creative video and broadcast applications.
  • LibreOffice is a free and open-source office productivity software suite, a project of The Document Foundation. It is a perfect alternative for the Microsoft Office suite.
  • FontSpace is a designer-centered font website that has quick customizable previews and hassle-free downloads. Every font is added and categorized by a real person. Each font is reviewed by a FontSpace moderator, checked for font quality issues, and licenses are verified.
  • Google Fonts is a library of 1474 open source font families and APIs for convenient use via CSS and Android.
  • Google Tasks is an app and add-on that lets you make to-do lists and set reminders. Your Google Tasks reminders are organized into lists, and can be tied to specific times and dates. I personally use the mobile app and it has helped a lot in my productivity
  • Kdenlive is a powerful free and open source cross-platform video editing program made by the KDE community.
  • Kenney is an amazing developer who has a whole store of free assets mainly focused on game development, but that you can use on any project you like.
Graphic Design
  • Photopea is an extremely helpful software pretty similar to Photoshop and it is completely free. You can save and open your works on PSD and export them to multiple other formats. The only inconvenient I've found with this is that the bigger your project, the laggier it becomes.
  • Paint.net is another excellent software used by many people. Similar to Paint but with a lot more features.
  • Gimp is very helpful and easy-to-learn for new artists. It is like an upgraded version of MS Paint.
  • Inkscape is one of the best alternatives if you want to create vectors. Pretty useful for UI designers and artists.
  • Gravit Designer is a free web alternative to Adobe Illustrator where you can design professional vectors without paying anything. Almost every single feature is free, but it has an optional monthly subscription if you want some more features
  • Figma is another excellent option for UI designers. It is free for beginners and offers a lot of options and tools to create amazing and beautiful looking UI for your games.
  • Krita is a free and open source tool created by artists. It is mainly used to draw, but it might be also helpful in other ways.
  • Vectr is a free graphics software used to create vector graphics easily and intuitively. It's a simple yet powerful web and desktop cross-platform tool
  • Game UI Database is a website where you can find inspiration for your UI designs. You can filter by type, style and many other filters.
  • Pixlr and was made specifically for photo manipulation, but you can still do some pretty basic GFX with it.
  • Adobe XD comes with a free starter plan and Lunacy which is similar to Adobe XD.
  • Boxicons is an amazing site where you can get a variety of icons for your projects. I'm currently using it in this same website.
Programming
  • Visual Studio Code is one of the most used code editors. It will help you keep your code organized and speed up your workflow.
  • GitHub is an amazing tool to help you use Git, a version conrol system. It provides the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.
  • GitHub Copilot is a cloud-based artificial intelligence tool developed by GitHub and OpenAI to assist users of Visual Studio Code, Visual Studio, Neovim, and JetBrains integrated development environments by autocompleting code. It is currently only available with a paid subscription, but you can apply for the GitHub Student Pack, and then it is free. Definetly worth a try.
  • freeCodeCamp is a friendly community where you can learn to code for free. It is run by a donor-supported 501(c)(3) nonprofit to help millions of busy adults transition into tech.
  • Codepen is an online community for testing and showcasing user-created HTML, CSS and JavaScript code snippets. It is a social development environment for front-end designers and developers.
  • Stack Overflow is a massive online Q&A community where developers help each other out with problems.
  • Atom is a very powerful editor which is modular and highly customizable. Since its free and open-source, it has strong community support and plethora of community-developed packages to extend its functionality.
Modelling, building and terrain
  • Blender is the free and open source 3D creation suite. It supports the entirety of the 3D pipeline—modeling, rigging, animation, simulation, rendering, compositing and motion tracking, even video editing and game creation.
  • Magica Voxel is probably the most popular voxel program among artists because it has a powerful GPU-based renderer built into it. This allows you to create stunning renders with lighting and material effects. It's free to use on any project and was created by ephtracy.
  • CC0 textures is a large library of free PBR materials (1100+ assets). Many of these assets have Substance files (. sbsar) available to download.
  • Poly Pizza is an amazing website full of free models created by the youtuber Dook and many other people. It is fully inspired on the previous Google's project, Poly, which was discontinued and no longer available.
  • ZBrush Core Mini is a digital sculpting application that is free for anyone to use. It is designed for new artists of all ages, with a streamlined interface that lets anyone jump right in and start sculpting immediately. Start with a simple sphere or a stone block, then use a selection of the most popular digital sculpting brushes from ZBrush to add or remove material and simply... create.
  • QuadSpinner Gaea is the industry's most loved terrain design application for VFX, games, and virtual production. The Gaea Community Edition is a free, fully-featured version with some limitations.
Music and SFX
  • Strofe is an A.I. that helps you make music that fits the mood and theme of your video game, Twitch stream, Youtube video, podcast and more.
  • BFXR is a generator for simple sound effects, created by DrPetter. You can use it to create easy retro sounds for remixing or video games.
  • Audacity is a free and open-source digital audio editor and recording application software
  • LMMS is a digital audio workstation application program. It allows music to be produced by arranging samples, synthesizing sounds, playing on a MIDI keyboard, and combining the features of trackers and sequencers.
  • Zapsplat is one of the fastest growing free sound effects & royalty free music libraries with thousands of professional, free sounds to download instantly in mp3 or original wav file format and upload hundreds more every week.
  • Bandlab Android Android Bandlab iOS are bith apps to create simple songs, mix and record with just your phone. The best part is that it also works as a social web to musicians.
